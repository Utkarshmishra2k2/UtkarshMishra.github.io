Project

AI-Powered Data Analytics Platform: Developed a platform leveraging Gemini LLM to dynamically generate Python code for real-time data insights and visualization. Integrated custom transformer models for enhanced context-aware analysis. Built with Streamlit & DataStax for seamless session management.

Uber Estimated Time of Arrival (ETA) Prediction: An AI-powered ETA prediction app using TensorFlow & Streamlit to provide real-time travel estimates based on live traffic data.

Study App: A Streamlit application that extracts, enhances, translates, and converts transcripts to audio. Utilizes transformers for summarization, Google Translate API for language conversion, and gTTS for audio generation.

Factor Analysis & PCA on Airline Data: This project applies factor analysis to extract underlying patterns, followed by PCA for dimensionality reduction. The workflow includes logistic regression for predictive modeling and K-Means clustering for segmentation.

Furniture Sales Prediction: A time-series forecasting model that helps businesses optimize inventory and marketing decisions using predictive analytics.

Crop Analysis: Built a recommendation system that suggests optimal crops based on environmental conditions such as temperature and humidity, supporting sustainable agriculture.
