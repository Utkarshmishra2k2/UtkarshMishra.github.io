Projects

Uber Estimated Time of Arrival (ETA) Prediction: This project leverages Streamlit for an interactive web interface and TensorFlow for machine learning to predict the Estimated Time of Arrival (ETA) for Uber rides. The model and application work together to provide real-time predictions and insightful data visualizations.

Study App: The Streamlit application leveraging various tools for transcript extraction, enhancement, translation, and audio conversion. It utilizes the transformers library to enhance transcript summarization, googletrans for translation, and gtts for audio conversion. The core functionality includes extracting transcripts from YouTube videos, summarizing them, translating them into different languages, and finally converting them into audio format.

Factor Analysis & PCA on Airline Data: This project explores different factor rotation methods, interprets the factors based on their loadings, and applies logistic regression to predict a binary target variable, followed by K Means clustering to identify patterns within the data.

Statistics In Python: A collection of advanced statistical techniques and data analysis tools implemented in Python.

Furniture Sales Prediction: By using time series models to analyze historical sales data, this project helps businesses make informed decisions about inventory, marketing, and operations.

Crop Analysis: This project builds a recommendation model for crop selection based on environmental factors to understand the impact of raising temperatures and reducing rainfall on crop viability.
